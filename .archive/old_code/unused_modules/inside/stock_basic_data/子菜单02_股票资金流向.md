# 股票资金流向功能说明

## 功能概述
股票资金流向功能展示各股票的资金流入流出情况，帮助分析资金动向。

## 数据来源
通过东方财富网资金流向接口获取数据。

## 数据获取流程
1. 每日开盘后执行`basic_data_other_daily_job.py`
2. 调用`stockfetch.fetch_stocks_fund_flow`获取不同时间段的资金流向数据
3. 合并今日、3日、5日、10日数据
4. 存储到`cn_stock_fund_flow`数据表

## 主要数据字段
- 今日数据：今日主力净流入-净额、今日主力净流入-净占比等
- 3日数据：3日主力净流入-净额、3日主力净流入-净占比等
- 5日数据：5日主力净流入-净额、5日主力净流入-净占比等
- 10日数据：10日主力净流入-净额、10日主力净流入-净占比等
- 按资金类型分类：超大单、大单、中单、小单

## 计算方法
- 主力资金 = 超大单资金 + 大单资金
- 散户资金 = 中单资金 + 小单资金
- 净流入 = 流入资金 - 流出资金
- 净占比 = 净流入 / 成交额 * 100%

## 更新频率
每个交易日上午开盘后更新一次。
