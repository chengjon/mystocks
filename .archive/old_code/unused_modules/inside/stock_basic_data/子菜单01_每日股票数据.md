# 每日股票数据功能说明

## 功能概述
每日股票数据是股票基本数据的核心组成部分，展示股票的实时行情信息。

## 数据来源
通过东方财富网接口获取股票实时行情数据。

## 数据获取流程
1. 每日开盘后执行`basic_data_daily_job.py`
2. 调用`stock_data`单例获取股票列表
3. 通过`stockfetch.fetch_stocks`从东方财富网获取数据
4. 存储到`cn_stock_spot`数据表

## 主要数据字段
- 基本信息：代码、名称、最新价、涨跌幅、涨跌额
- 价格信息：今开、最高、最低、昨收
- 成交信息：成交量、成交额、振幅、换手率、量比
- 财务指标：市盈率、市净率、每股收益、每股净资产等
- 市值信息：总市值、流通市值

## 计算方法
- 涨跌幅 = (最新价 - 昨收) / 昨收 * 100%
- 振幅 = (最高 - 最低) / 昨收 * 100%
- 换手率 = 成交量 / 流通股本 * 100%
- 量比 = 当前成交量 / 前5日平均成交量

## 更新频率
每个交易日上午开盘后更新一次。
