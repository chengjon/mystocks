# 尾盘抢筹数据功能说明

## 功能概述
尾盘抢筹数据功能展示收盘竞价期间资金抢筹的股票，反映收盘资金活跃度。

## 数据来源
通过通达信TQLEX接口获取尾盘竞价抢筹数据。

## 数据获取流程
1. 每日收盘后执行相关任务
2. 调用`stock_chip_race.stock_chip_race_end`获取数据
3. 存储到`cn_stock_chip_race_end`数据表

## 主要数据字段
- 股票代码和名称
- 最新价和涨跌幅
- 昨收和今开价格
- 收盘金额
- 抢筹幅度
- 抢筹委托金额
- 抢筹成交金额
- 抢筹占比

## 计算方法
- 抢筹幅度 = (抢筹成交金额 - 抢筹委托金额) / 抢筹委托金额 * 100%
- 抢筹占比 = 抢筹成交金额 / 收盘金额 * 100%
- 涨跌幅 = (最新价 - 昨收) / 昨收 * 100%

## 更新频率
每个交易日收盘后更新一次。
