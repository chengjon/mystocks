# 股票策略数据功能说明

## 功能概述

股票策略数据功能是本项目的核心功能之一，用于识别和筛选符合特定技术策略的股票。该功能通过每日自动计算，筛选出符合预设策略条件的股票，并将结果存储在对应的数据库表中。

## 功能实现流程

### 1. 策略计算流程
1. 每日通过`strategy_data_daily_job.py`计算所有股票的策略条件
2. 并行调用各个策略的检查函数
3. 将符合条件的股票存储到对应的策略数据表中

### 2. 策略列表
系统内置了10种不同的股票策略，每种策略对应一个独立的数据表：

1. 放量上涨 (cn_stock_strategy_enter)
2. 均线多头 (cn_stock_strategy_keep_increasing)
3. 停机坪 (cn_stock_strategy_parking_apron)
4. 回踩年线 (cn_stock_strategy_backtrace_ma250)
5. 突破平台 (cn_stock_strategy_breakthrough_platform)
6. 无大幅回撤 (cn_stock_strategy_low_backtrace_increase)
7. 海龟交易法则 (cn_stock_strategy_turtle_trade)
8. 高而窄的旗形 (cn_stock_strategy_high_tight_flag)
9. 放量跌停 (cn_stock_strategy_climax_limitdown)
10. 低ATR成长 (cn_stock_strategy_low_atr)

## 关联项目文件

### 核心文件
- `instock/job/strategy_data_daily_job.py` - 策略数据计算任务
- `instock/core/tablestructure.py` - 策略表结构定义
- `instock/core/strategy/` - 各策略具体实现文件

### 策略实现文件
- `instock/core/strategy/enter.py` - 放量上涨策略
- `instock/core/strategy/keep_increasing.py` - 均线多头策略
- `instock/core/strategy/parking_apron.py` - 停机坪策略
- `instock/core/strategy/backtrace_ma250.py` - 回踩年线策略
- `instock/core/strategy/breakthrough_platform.py` - 突破平台策略
- `instock/core/strategy/low_backtrace_increase.py` - 无大幅回撤策略
- `instock/core/strategy/turtle_trade.py` - 海龟交易法则策略
- `instock/core/strategy/high_tight_flag.py` - 高而窄的旗形策略
- `instock/core/strategy/climax_limitdown.py` - 放量跌停策略
- `instock/core/strategy/low_atr.py` - 低ATR成长策略

## 数据表结构

每个策略对应一个独立的数据表，表名格式为`cn_stock_strategy_xxx`，包含以下字段：
- 基础字段：date(日期)、code(代码)、name(名称)、open(开盘价)、high(最高价)、low(最低价)、close(收盘价)等
- 回测数据字段：用于后续的策略回测分析

## Web展示
策略数据通过Web界面的"股票策略数据"菜单项展示，用户可以查看每日符合各策略条件的股票列表。
