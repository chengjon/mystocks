# 股票策略计算方法详解

## 1. 放量上涨策略 (enter.py)

### 策略条件
1. 当日比前一天上涨小于2%或收盘价小于开盘价
2. 当日成交额不低于2亿
3. 当日成交量/5日平均成交量>=2

### 计算方法
- 检查当日涨幅是否小于2%或收盘价是否小于开盘价
- 计算当日成交额(收盘价×成交量)是否不低于2亿
- 计算5日平均成交量，检查当日成交量是否是5日平均成交量的2倍以上

## 2. 均线多头策略 (keep_increasing.py)

### 策略条件
1. 30日前的30日均线 < 20日前的30日均线 < 10日前的30日均线 < 当日的30日均线
2. (当日的30日均线/30日前的30日均线) > 1.2

### 计算方法
- 计算30日移动平均线
- 检查最近30个交易日中，30日均线是否呈现持续上升趋势
- 检查区间内最大涨幅是否超过20%

## 3. 停机坪策略 (parking_apron.py)

### 策略条件
1. 最近15日有涨幅大于9.5%，且必须是放量上涨
2. 紧接的下个交易日必须高开，收盘价必须上涨，且与开盘价不能大于等于相差3%
3. 接下2、3个交易日必须高开，收盘价必须上涨，且与开盘价不能大于等于相差3%，且每天涨跌幅在5%间

### 计算方法
- 在最近15个交易日中寻找涨停日(涨幅>9.5%)
- 检查涨停日是否为放量上涨
- 检查涨停后3个交易日是否形成停机坪形态

## 4. 回踩年线策略 (backtrace_ma250.py)

### 策略条件
1. 时间段：前段=最近60交易日最高收盘价之前交易日(长度>0)，后段=最高价当日及后面的交易日
2. 前段由年线(250日)以下向上突破
3. 后段必须在年线以上运行，且后段最低价日与最高价日相差必须在10-50日间
4. 回踩伴随缩量：最高价日交易量/后段最低价日交易量>2,后段最低价/最高价<0.8

### 计算方法
- 计算250日移动平均线(年线)
- 在最近60个交易日中找出最高价和最低价
- 划分前后两段，检查前段是否由年线以下向上突破
- 检查后段是否在年线以上运行
- 检查时间间隔和成交量变化是否符合要求

## 5. 突破平台策略 (breakthrough_platform.py)

### 策略条件
1. 60日内某日收盘价>=60日均线>开盘价
2. 且【1】放量上涨
3. 且【1】间之前时间，任意一天收盘价与60日均线偏离在-5%~20%之间

### 计算方法
- 计算60日移动平均线
- 寻找满足收盘价>=60日均线>开盘价的交易日
- 检查该日是否为放量上涨
- 检查该日前所有交易日的收盘价与60日均线的偏离度

## 6. 无大幅回撤策略 (low_backtrace_increase.py)

### 策略条件
1. 当日收盘价比60日前的收盘价的涨幅小于0.6
2. 最近60日，不能有单日跌幅超7%、高开低走7%、两日累计跌幅10%、两日高开低走累计10%

### 计算方法
- 计算60日内的价格涨幅
- 检查是否存在大幅回撤的情况
- 允许有一次"洗盘"行为

## 7. 海龟交易法则策略 (turtle_trade.py)

### 策略条件
最后一个交易日收市价为指定区间内最高价
1. 当日收盘价>=最近60日最高收盘价

### 计算方法
- 找出最近60个交易日的最高收盘价
- 检查当日收盘价是否等于或超过历史最高价

## 8. 高而窄的旗形策略 (high_tight_flag.py)

### 策略条件
1. 必须至少上市交易60日
2. 当日收盘价/之前24~10日的最低价>=1.9
3. 之前24~10日必须连续两天涨幅大于等于9.5%
4. 龙虎榜上必须有机构

### 计算方法
- 检查股票上市时间是否满足要求
- 计算24~10日前的最低价
- 检查价格涨幅是否满足要求
- 检查是否连续两天涨停
- 检查是否在龙虎榜上有机构

## 9. 放量跌停策略 (climax_limitdown.py)

### 策略条件
1. 跌>9.5%
2. 成交额不低于2亿
3. 成交量至少是5日平均成交量的4倍

### 计算方法
- 检查当日跌幅是否超过9.5%
- 计算当日成交额是否不低于2亿
- 计算5日平均成交量，检查当日成交量是否是5日平均成交量的4倍以上

## 10. 低ATR成长策略 (low_atr.py)

### 策略条件
1. 必须至少上市交易250日
2. 最近10个交易日的最高收盘价必须比最近10个交易日的最低收盘价高1.1倍

### 计算方法
- 检查股票上市时间是否满足要求
- 计算最近10个交易日的最高价和最低价
- 检查价格波动幅度是否满足要求
- 计算ATR(平均真实波幅)是否较低
