# 概念资金流向功能说明

## 功能概述
概念资金流向功能展示各概念板块的资金流入流出情况，帮助分析资金在不同概念间的流动。

## 数据来源
通过东方财富网概念资金流向接口获取数据。

## 数据获取流程
1. 每日开盘后执行相关任务
2. 调用概念资金流向接口获取数据
3. 存储到`cn_stock_fund_flow_concept`数据表

## 主要数据字段
- 概念名称
- 概念涨跌幅
- 主力净流入-净额
- 主力净流入-净占比
- 各类型资金流向：超大单、大单、中单、小单

## 计算方法
- 主力资金 = 超大单资金 + 大单资金
- 散户资金 = 中单资金 + 小单资金
- 净流入 = 流入资金 - 流出资金
- 净占比 = 净流入 / 概念总成交额 * 100%

## 更新频率
每个交易日上午开盘后更新一次。
