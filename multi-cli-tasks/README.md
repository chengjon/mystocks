# Multi-CLI Worktree Operations Center

欢迎来到多 CLI 协作运营中心。这是本项目管理并行开发生命周期的指挥舱。

## 🏢 目录架构

- **[guides/](./guides/)**: 核心规范与手册
  - **[MULTI_CLI_WORKTREE_MANAGEMENT.md](./guides/MULTI_CLI_WORKTREE_MANAGEMENT.md)**: 体系总纲 (核心)
  - **[MAIN_CLI_WORKFLOW.md](./guides/MAIN_CLI_WORKFLOW.md)**: 主CLI必读操作手册
  - **[WORKER_CLI_GUIDE.md](./guides/WORKER_CLI_GUIDE.md)**: WorkerCLI必读执行指南
- **[guides/templates/](./guides/templates/)**: 标准化任务书与报告模板
- **mystocks_phaseX_.../**: 各个活跃或已结项的任务单元子目录。

## 🕹️ 快速操作

1.  **查阅标准**: 请进入 `guides/` 查阅对应的标准文档。
2.  **管理任务**: 请进入对应的worktree目录查看 `TASK.md` 和 `TASK-REPORT.md`。
3.  **阶段总结**: 查阅根目录下的 `PHASE_X_SUMMARY.md`, 由主CLI根据各个worktree目录下的 `TASK.md` 和 `TASK-REPORT.md`整合编辑。
