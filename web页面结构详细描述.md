# MyStocks 量化交易平台 - Web页面结构详细描述

## 概述

MyStocks是一个现代化的量化交易平台，采用Vue 3 + TypeScript + Element Plus技术栈，结合FastAPI后端，提供完整的量化交易、策略回测、实时监控等功能。整体设计风格为专业金融风，深色主题优先，数据可视化为核心。

---

## 1. 主仪表盘页面 (Dashboard.vue)

### 整体布局
- **顶部标题栏**：页面标题"仪表盘"，副标题"实时市场概览与投资组合监控"，右侧智能数据源指示器
- **统计卡片行**：4个统计卡片，响应式布局（xs:24, sm:12, md:6）
- **主要内容区**：左侧市场热度中心图表，右侧资金流向图表
- **底部表格区**：板块表现表格，包含4个标签页

### 详细元素描述

#### 1.1 顶部标题栏
- **左侧区域**：
  - 主标题：h1标签，字体大小24px，字重600，颜色#303133
  - 副标题：p标签，字体大小14px，颜色#909399
- **右侧区域**：
  - SmartDataIndicator组件：显示数据源模式（模拟/真实/混合）和健康状态

#### 1.2 统计卡片行
- **卡片容器**：el-row，gutter为20，底部外边距20px
- **单个卡片**：el-card，hover效果（向上移动5px，阴影加深）
- **卡片内容**：
  - **图标区域**：56x56px圆角矩形，白色图标，动态背景色
  - **信息区域**：
    - 标题：14px，颜色#909399，底部间距8px
    - 数值：24px，粗体，颜色#303133，底部间距8px
    - 趋势：12px，根据涨跌显示不同颜色（绿/红/灰）

#### 1.3 市场热度中心（左侧）
- **卡片头部**：
  - 标题"市场热度中心"
  - 右侧"重试"按钮，warning类型，小尺寸，loading状态
- **标签页切换**：el-tabs，包含4个标签
  - 市场热度（heat）：柱状图，展示指数热度
  - 领涨板块（leading）：柱状图，显示板块涨幅
  - 市场分布（distribution）：环形图，展示涨跌分布
  - 资金流向（capital）：柱状图，显示资金流向
- **图表区域**：350px高度，ECharts实现

#### 1.4 资金流向（右侧）
- **卡片头部**：
  - 标题"资金流向"
  - 下拉选择器：证监会/申万一级/申万二级，120px宽度
- **图表区域**：400px高度，横向柱状图，红绿配色

#### 1.5 板块表现表格（底部）
- **卡片头部**：
  - 标题"板块表现"
  - 右侧按钮组："刷新"和"重试"按钮
- **标签页**：el-tabs，4个标签页
  - **自选股**：表格显示自选股票信息
  - **策略选股**：显示策略推荐的股票
  - **行业选股**：按行业分类的股票
  - **概念选股**：热门概念相关股票
- **表格结构**：
  - 列：代码、名称、现价、涨跌幅、成交量/换手率/行业/策略/评分等
  - 涨跌幅颜色：红色上涨，绿色下跌
  - 最大高度400px，条纹样式

- **容器**：16px高度，12px顶部内边距

## 7. 智能数据指示器组件 (SmartDataIndicator.vue)

### 组件结构
- **模式指示器**：显示当前数据源模式
- **健康指示器**：显示系统健康状态
- **详细信息弹窗**：悬停显示详细状态

### 详细元素描述

#### 7.1 模式指示器
- **图标**：根据模式显示不同SVG图标
- **模式类型**：
  - mock：模拟数据（灰色）
  - real：真实数据（绿色）
  - hybrid：混合模式（蓝色）
  - unknown：检测中（橙色）

#### 7.2 健康指示器
- **状态点**：动画脉冲效果
- **状态颜色**：健康（绿）/异常（红）/检测中（橙）

#### 7.3 状态弹窗
- **信息行**：模式、健康状态、降级启用、最后检查时间
- **数据源详情**：各数据源状态、响应时间
- **响应式**：移动端适配

## 通用设计规范

### 颜色方案
- **主色调**：#409EFF（蓝色）
- **成功色**：#67C23A（绿色）
- **警告色**：#E6A23C（橙色）
- **危险色**：#F56C6C（红色）
- **信息色**：#909399（灰色）
- **涨跌颜色**：涨红(#f56c6c) / 跌绿(#67c23a)

### 字体规范
- **标题**：24-28px，字重600
- **副标题**：14px，灰色
- **正文**：14px
- **小字**：12-13px
- **数字**：等宽字体（Monaco/Menlo）

### 间距规范
- **页面边距**：20px
- **卡片间距**：16-20px
- **元素间距**：8-12px
- **内容内边距**：16-20px

### 响应式设计
- **断点**：xs(<768px), sm(768px+), md(992px+), lg(1200px+)
- **布局适配**：使用Element Plus的栅格系统
- **移动端优化**：表格最大高度、按钮堆叠、字体缩放

### 交互效果
- **悬停效果**：卡片阴影加深、按钮变色
- **加载状态**：loading动画、骨架屏
- **过渡动画**：0.2-0.3s ease过渡
- **微交互**：脉冲动画、渐变效果

## 数据可视化规范

### 图表类型
- **K线图**：candlestick，红绿配色
- **折线图**：平滑曲线，面积渐变填充
- **柱状图**：横向/纵向，颜色区分正负
- **饼图/环形图**：支持标签和百分比显示
- **散点图**：用于相关性分析

### 图表配置
- **工具提示**：cross类型，格式化显示
- **图例**：底部或右侧显示
- **网格**：合理的边距设置
- **缩放**：支持inside和slider缩放
- **响应式**：窗口resize自适应

## 特殊功能说明

### SSE实时推送
- **协议**：Server-Sent Events
- **通道**：训练、回测、告警、仪表板
- **状态监控**：连接数、健康状态
- **自动重连**：断线重连机制

### 智能数据源
- **模式切换**：自动/手动模式切换
- **降级策略**：真实数据失败时降级到模拟数据
- **健康检查**：定期检查数据源状态
- **缓存机制**：智能缓存提升性能

### 主题支持
- **默认主题**：浅色专业风格
- **深色主题**：深色背景，高对比度
- **自定义主题**：支持CSS变量覆盖
