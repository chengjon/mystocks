## 🚀 快速开始

### 环境准备

#### 必备工具清单
- **Python**: 3.12+ (推荐使用Anaconda或pyenv管理版本)
- **Node.js**: 18+ (推荐使用nvm管理版本)
- **Git**: 版本控制工具
- **Docker**: 容器化部署
- **VS Code**: 推荐的IDE（可选）

#### 开发环境安装
```bash
# 1. Python环境配置
# 使用pyenv管理Python版本
curl https://pyenv.run | bash
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init -)"' >> ~/.bashrc

# 安装Python 3.12
pyenv install 3.12.11
pyenv global 3.12.11

# 2. Node.js环境配置
# 使用nvm管理Node.js版本
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc

# 安装Node.js 18
nvm install 18
nvm use 18
nvm alias default 18

# 3. 项目环境初始化
git clone git@github.com:chengjon/mystocks.git
cd mystocks_spec

# Python虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
pip install -r web/backend/requirements.txt

# 前端依赖
cd web/frontend
npm install
```

## 🤖 Claude Code 集成

### 📋 Claude Code Hooks 系统概述

Claude Code Hooks 是 MyStocks 项目的自动化开发工具链，提供7个生产就绪的自动化脚本，实现开发、测试、部署全流程自动化。

### 🔧 Hooks 配置

#### 1. 技能激活 Hook
```bash
.claude/hooks/user-prompt-submit-skill-activation.sh
```
**功能**: 根据用户输入自动激活相应的开发技能
**使用场景**: 自动识别开发任务类型，调用对应的工具链

#### 2. 文件编辑追踪 Hook
```bash
.claude/hooks/post-tool-use-file-edit-tracker.sh
```
**功能**: 记录所有文件编辑操作，生成变更日志
**使用场景**: 追踪代码变更，生成提交信息

#### 3. 数据库架构验证 Hook
```bash
.claude/hooks/post-tool-use-database-schema-validator.sh
```
**功能**: 自动验证数据库架构变更的合理性
**使用场景**: 数据库表结构变更时的自动验证

#### 4. 文档整理 Hook
```bash
.claude/hooks/post-tool-use-document-organizer.sh
```
**功能**: 自动整理文档结构，确保文档组织规范
**使用场景**: 文档创建和更新时的自动整理

#### 5. 代码质量门禁 Hook
```bash
.claude/hooks/stop-python-quality-gate.sh
```
**功能**: 在代码提交前执行质量检查，确保代码质量
**使用场景**: 代码提交前的质量把关

#### 6. 任务管理 Hook
```bash
.claude/hooks/session-start-task-master-injector.sh
```
**功能**: 会话开始时注入任务上下文，提供开发指导
**使用场景**: 新开发会开始时的任务初始化

#### 7. 会话清理 Hook
```bash
.claude/hooks/session-end-cleanup.sh
```
**功能**: 会话结束时清理临时文件，整理工作环境
**使用场景**: 开发会结束时的环境清理

### 🎯 Playwright 高级功能

#### 1. 页面对象模型
```typescript
// tests/page-objects/StockDashboard.ts
import { Page, Locator } from '@playwright/test';

export class StockDashboard {
  readonly page: Page;
  readonly stockList: Locator;
  readonly searchInput: Locator;
  readonly realTimePrice: Locator;

  constructor(page: Page) {
    this.page = page;
    this.stockList = page.locator('[data-testid="stock-list"]');
    this.searchInput = page.locator('[data-testid="stock-search"]');
    this.realTimePrice = page.locator('[data-testid="real-time-price"]');
  }

  async goto() {
    await this.page.goto('/');
    await this.stockList.waitFor();
  }

  async searchStock(symbol: string) {
    await this.searchInput.fill(symbol);
    await this.page.keyboard.press('Enter');
    await this.page.locator('[data-testid="search-results"]').waitFor();
  }

  async getStockPrice(symbol: string) {
    const priceElement = this.page.locator(`[data-testid="stock-price-${symbol}"]`);
    return await priceElement.textContent();
  }

  async waitForRealTimeUpdate() {
    await this.realTimePrice.waitFor({ state: 'visible' });
  }
}
```

```typescript
// tests/performance.spec.ts
import { test, expect } from '@playwright/test';

test.describe('性能测试', () => {
  test('页面加载性能', async ({ page }) => {
    const startTime = Date.now();
    await page.goto('/');
    const loadTime = Date.now() - startTime;

    // 页面加载时间应小于3秒
    expect(loadTime).toBeLessThan(3000);

    // 检查Core Web Vitals
    const vitals = await page.evaluate(() => {
      return new Promise((resolve) => {
        new PerformanceObserver((list) => {
          const entries = list.getEntries();
          const navigation = entries.find(entry => entry.entryType === 'navigation');
          resolve(navigation);
        }).observe({ entryTypes: ['navigation'] });
      });
    });

    expect(vitals.loadEventEnd).toBeLessThan(3000);
  });

  test('大数据量渲染性能', async ({ page }) => {
    await page.goto('/');

    // 模拟加载1000条股票数据
    await page.evaluate(() => {
      window.loadLargeDataset = () => {
        const stockList = document.querySelector('[data-testid="stock-list"]');
        for (let i = 0; i < 1000; i++) {
          const item = document.createElement('div');
          item.setAttribute('data-testid', `stock-item-${i}`);
          item.textContent = `股票${i}`;
          stockList.appendChild(item);
        }
      };
    });

    const startTime = Date.now();
    await page.evaluate(() => window.loadLargeDataset());
    const renderTime = Date.now() - startTime;

    // 渲染时间应小于1秒
    expect(renderTime).toBeLessThan(1000);
  });
```

```yaml
# .github/workflows/ci.yml
name: CI/CD Pipeline

on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  test:

    - name: Run tests
      run: |
        pytest tests/ -v --cov=src --cov-report=xml

    - name: Upload coverage
      uses: codecov/codecov-action@v3

  frontend-test:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v3

    - name: Set up Node.js
      uses: actions/setup-node@v3

        node-version: '18'

    - name: Install dependencies
      working-directory: ./web/frontend
      run: npm ci

    - name: Run tests
      working-directory: ./web/frontend
      run: npm test

    - name: Run E2E tests
      working-directory: ./web/frontend

    - uses: actions/checkout@v3
```

## 📞 技术支持

### 🆘 常见问题解决

#### 1. Claude Code Hook 失败
```bash
# 检查Hook权限
ls -la .claude/hooks/
chmod +x .claude/hooks/*.sh

# 查看Hook日志
tail -f .claude/logs/hooks.log

# 重新安装Hooks
.claude/hooks/reinstall.sh
```

#### 2. APIFox 同步问题
```bash
# 检查网络连接
ping api.apifox.cn

# 重新生成API Token
# 1. 登录APIFox
# 2. 进入项目设置
# 3. 重新生成Token
# 4. 更新环境变量

# 手动同步接口
curl -X POST \
  -H "Authorization: Bearer $APIFOX_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"openapi": "$(cat openapi.json)"}' \
  https://api.apifox.cn/v1/projects/$PROJECT_ID/import
```

#### 3. Playwright 测试失败
```bash
# 更新浏览器
npx playwright install

# 检查测试环境
npx playwright test --list

# 调试模式运行
npx playwright test --debug

# 生成测试报告
npx playwright test --reporter=html
```
