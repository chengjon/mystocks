## 使用说明

### 查找特定API

1. 根据API路径前缀（如 `/auth`, `/data`, `/dashboard`）找到对应文件
2. 点击链接打开对应的JSON文件
3. 在文件中查找具体的API端点

### 快速导航

```bash
# 查看所有文件
ls -lh docs/reports/api_split/

# 查看特定文件
cat docs/reports/api_split/api_auth.json

# 搜索特定API
grep -r "/login" docs/reports/api_split/
```

### 文件说明

- **`api_index.json`**: 索引文件，包含所有拆分文件的元数据
- **`api_*.json`**: 按路径前缀拆分的API数据文件
- **`API_SPLIT_INDEX.md`**: 本文档，提供友好的索引界面

### 数据格式

每个拆分文件包含以下字段：

```json
{
  "generated_at": "2026-01-02T00:32:22.264253",
  "prefix": "/auth",
  "total_endpoints": 5,
  "endpoints": [
    {
      "path": "/login",
      "method": "POST",
      "file": "auth.py",
      "function": "login_for_access_token",
      "return_model": "dict",
      "data_fields": [],
      "db_dependencies": [],
      "source_type": "postgresql",
      "line_number": 172
    }
  ]
}
```
