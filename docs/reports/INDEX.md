## Historical Snapshot Scope

- `docs/reports/` 根目录下的大量 markdown 仍保留为历史报告、计划、审计和复盘快照
- 文内数字、完成状态、验证结论和优先级如未重新复核，应按文档日期理解为历史观察，不得直接当作当前事实
- 后续治理应继续做 bounded archive、分类收口和边界说明补齐，而不是把本索引重新扩写成扁平文件总表
