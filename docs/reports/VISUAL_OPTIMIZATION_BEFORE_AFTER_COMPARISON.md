## 🔴 优化前：问题诊断 (Before)

### 问题1: 按钮文字对齐混乱 (31/31页面受影响)

**严重程度**: 🔴 P0 - 最高优先级（影响用户交互体验）

**具体表现**:
```
❌ 按钮padding不一致:
   - Dashboard: padding: 2px 8px, 4px 16px, 12px 24px（至少5种）
   - Market: padding: 8px 16px, 10px 20px（至少3种）
   - Stocks: padding: 6px 12px, 8px 16px, 10px 20px（至少4种）

❌ 文字对齐偏移:
   - 部分按钮文字偏左（缺少justify-content: center）
   - 部分按钮文字偏右（text-align: right）
   - 部分按钮文字偏上（缺少align-items: center）
   - 部分按钮文字偏下（line-height > 1）

❌ 按钮高度混乱:
   - 小按钮: 28px, 30px, 32px（至少3种）
   - 标准按钮: 36px, 38px, 40px（至少3种）
   - 大按钮: 44px, 48px, 50px（至少3种）
```

**影响页面**: 全部31个页面
- P0核心页面: 6/6个（100%）
- P1重要页面: 8/8个（100%）
- P2辅助页面: 17/17个（100%）

**用户影响**:
- 🔴 **严重**: 按钮交互感差，点击区域不精确
- 🔴 **严重**: 界面显得不专业，降低品牌信任度
- 🔴 **严重**: 视觉混乱，影响用户体验

**技术根因**:
```scss
// 问题代码示例（Element Plus默认值）
.el-button {
  display: inline-block;  // ❌ 无法垂直居中
  padding: 12px 20px;    // ❌ padding不统一
  line-height: 1.5;      // ❌ 导致文字偏移
  text-align: left;      // ❌ 水平不居中
}
```

### 问题3: 组件间距混乱 (31/31页面受影响)

**严重程度**: 🟡 P2 - 中等优先级（影响视觉整洁度）

**具体表现**:
```
❌ 间距使用非8px倍数:

   - 组件内间距: 4px ~ 20px（范围过大）

   - 模块间间距: 20px ~ 60px（范围过大）

❌ 同页面内差异过大:
   - Dashboard: margin混合使用 7px, 12px, 20px, 30px
   - Market: padding混合使用 10px, 16px, 24px
   - Stocks: gap混合使用 12px, 16px, 20px
```

**影响页面**: 全部31个页面
- P0核心页面: 6/6个（100%）
- P1重要页面: 8/8个（100%）
- P2辅助页面: 17/17个（100%）

- 🟡 **轻微**: 增加认知负荷
- 🟡 **轻微**: 降低视觉舒适度

**技术根因**:
```scss
// 问题代码示例（常见不规范间距）

  margin: 15px 20px;     // ❌ 非规范值

.data-grid {
  gap: 17px;             // ❌ 非规范值
```
