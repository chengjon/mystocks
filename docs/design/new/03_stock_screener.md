# 03 智能选股池 (Stock Screener)

## 1. 页面概览
**功能定位**: 发现交易机会的源头。管理股票池，并基于技术/基本面指标筛选标的。
**视觉焦点**: 清晰的表格数据与顶部的筛选条件构造器。

## 2. 布局结构

### 2.1 顶部：股票池切换与统计 (Pool Selector)
*   **组件**: `PoolTabs`
*   **内容**:
    *   **全市场**: A股所有标的。
    *   **核心池**: 沪深300 / 中证500 / 创业板指。
    *   **行业池**: 按申万一级行业分类。
    *   **概念池**: 热门概念 (如: 新能源, 芯片)。
    *   **自定义池**: 用户创建的静态或动态池。
*   **统计徽标**: 每个Tab旁显示当前池内股票数量。

### 2.2 筛选控制栏 (Filter Builder)
*   **组件**: `ConditionBuilder`
*   **功能**:
    *   **快速筛选**: 下拉选择常用条件 (e.g., "放量突破", "MACD金叉", "低市盈率").
    *   **自定义条件**:
        *   指标: PE, PB, 市值, 涨幅, 换手率, MA均线状态.
        *   逻辑: >, <, =, 区间.
        *   数值: 输入具体数值.
    *   **执行按钮**: "开始选股" (ArtDeco 风格金色按钮).
*   **API**:
    *   虽然API文档未明确列出通用筛选接口，设计上假设后端支持或基于 `GET /api/v1/market/quotes` 返回的丰富字段在前端进行过滤。
    *   理想接口: `POST /api/v1/market/scan` (提交筛选条件)

### 2.3 结果列表 (Results Table)
*   **组件**: `SmartTable`
*   **列配置**:
    *   基础: 代码, 名称, 现价, 涨跌幅, 换手率, 量比.
    *   财务: PE(TTM), PB, 总市值, 流通市值.
    *   技术: 5日涨幅, 20日涨幅, 乖离率.
    *   **操作列**: 加入自选, 启动回测, 查看详情.
*   **交互**:
    *   表头点击排序。
    *   右键菜单: 批量操作 (批量加入自选, 批量导出CSV).
    *   行样式: 隔行变色 (深灰/黑)，鼠标悬停高亮 (金色边框).

## 3. 股票池管理 (Pool Management)
*   **侧边抽屉 (Drawer)**: 点击 "管理股票池" 打开。
*   **功能**:
    *   新建股票池 (静态: 手动导入; 动态: 保存筛选条件).
    *   编辑/删除现有池.
    *   查看池内变更记录 (调入/调出).

## 4. API 映射汇总
| 组件区域 | 核心 API 端点 | 请求方式 | 备注 |
| :--- | :--- | :--- | :--- |
| 行情数据 | `/api/v1/market/quotes` | GET | 支持批量获取 |
| 行业/概念 | `/api/v1/market/chip-race` | GET | 获取板块列表 |
| 基础数据 | `/api/v1/data/stock-basic` | GET | 获取股本/上市日期等 |
