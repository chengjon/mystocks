# 04 数据深度分析 (Data Analysis)

## 1. 页面概览
**功能定位**: 量化研究的核心阵地。提供比常规看盘更深度的统计分析、因子有效性检验和宏观数据可视化。
**视觉焦点**: 复杂图表的大屏展示 (热力图、散点图、3D曲面图)。

## 2. 布局结构 (Dashboard Layout)

页面由多个独立但关联的分析模块组成，支持拖拽布局。

### 2.1 模块一：市场全景热力 (Market Heatmap)
*   **组件**: `TreeMapWidget` (矩形树图)
*   **内容**:
    *   按行业/概念分组展示全市场股票。
    *   **矩形面积**: 总市值或成交额。
    *   **颜色**: 涨跌幅 (红涨绿跌)。
    *   **交互**: 滚轮缩放查看细分行业，点击进入行业详情。
*   **数据源**: `GET /api/v1/market/quotes` (全量) + 行业分类数据。

### 2.2 模块二：因子IC分析 (IC Analysis)
*   **组件**: `ICChart`
*   **内容**:
    *   **IC时序图**: 选定因子 (如 PE, Momentum) 在过去一段时间的 Rank IC 值走势。
    *   **IC衰减图**: 因子预测未来 1d, 5d, 20d 收益的能力衰减情况。
*   **API**:
    *   `GET /api/v1/analysis/ic` (假设有此分析接口，或基于策略回测接口 `strategy/results` 的因子部分)。

### 2.3 模块三：筹码分布与博弈 (Chip Distribution)
*   **组件**: `ChipRaceChart`
*   **内容**:
    *   展示获利盘比例、平均成本、筹码集中度。
    *   **火焰山图**: 筹码在不同价格区间的分布堆积。
*   **数据源**: `GET /api/v1/market/chip-race`

### 2.4 模块四：龙虎榜透视 (Dragon Tiger Lens)
*   **组件**: `LhbTable`
*   **内容**:
    *   展示今日上榜个股及其实力游资/机构动向。
    *   **关联分析**: 某知名游资近期共同操作的股票。
*   **数据源**: `GET /api/v1/market/lhb`

## 3. 交互设计
*   **参数面板**: 侧边栏可展开参数面板，调整分析的时间窗口、标的范围、使用的因子。
*   **图表联动**: 在热力图中点击某行业，IC分析图自动切换为该行业内的因子有效性分析。
