# 01 主控仪表盘 (Dashboard)

## 1. 页面概览
**功能定位**: 系统的"舰桥"，提供市场全局状态、核心资产概况和紧急事项的即时视图。
**视觉焦点**: 突出关键数字（红涨绿跌），使用金色边框强调核心指数。

## 2. 布局结构 (Grid Layout)

页面分为三层结构：
1.  **市场概况层 (Top)**: 核心指数与市场状态。
2.  **主力监控层 (Middle)**: 资金流向与涨跌分布。
3.  **策略与预警层 (Bottom)**: 实时预警与策略表现。

### 2.1 顶部：核心指数 (Market Indices)
*   **组件**: `IndexCard` (x3)
*   **数据源**: `GET /api/v1/market/quotes?symbols=000001.SH,399001.SZ,399006.SZ`
*   **展示内容**:
    *   **上证指数**: 当前点位 (大号字体), 涨跌幅, 成交额。
    *   **深证成指**: 同上。
    *   **创业板指**: 同上。
    *   **微型走势图**: 使用 Sparkline (迷你折线图) 展示当日分时走势。
*   **交互**: 点击卡片跳转至 [市场行情中心] 并聚焦该指数。
*   **ArtDeco细节**: 卡片使用双层金线边框，数字背景使用极淡的金色渐变。

### 2.2 中部左侧：市场温度计 (Market Breadth)
*   **组件**: `MarketThermometer`
*   **数据源**: `GET /api/v1/market/quotes` (聚合计算) 或 `GET /api/v1/market/distribution` (需确认后端支持或前端计算)
*   **展示内容**:
    *   **涨跌分布柱状图**: 涨停, >7%, >5%, 0-5%, 平, 0--5%, <-5%, <-7%, 跌停。
    *   **上涨/下跌家数对比**: 动态数字展示 (如: 3500 涨 / 1200 跌)。
    *   **赚钱效应**: 综合评分 (0-100)，仪表盘样式展示。

### 2.3 中部右侧：资金流向雷达 (Fund Flow Radar)
*   **组件**: `FundFlowChart`
*   **数据源**: `GET /api/v1/market/fund-flow`
*   **展示内容**:
    *   **北向资金**: 沪股通/深股通 实时净流入曲线。
    *   **主力资金**: 行业板块主力净流入 Top 5 列表 (水平条形图)。

### 2.4 底部左侧：实时预警终端 (Alert Terminal)
*   **组件**: `LiveAlertFeed`
*   **数据源**: `WebSocket /api/v1/monitoring/realtime` (Topic: alerts) 或 `GET /api/v1/monitoring/alerts?status=active`
*   **展示内容**:
    *   滚动列表，显示最新产生的系统预警或市场信号。
    *   **格式**: `[时间] [级别-图标] [标的] 消息内容`
    *   **示例**: `10:30:05 ⚠️ [Strategy-A] 600519.SH 触发止损信号`
*   **交互**: 点击条目跳转至 [风险监控中心] 或 [交易台]。

### 2.5 底部右侧：策略表现概览 (Strategy Performance)
*   **组件**: `StrategyMiniTable`
*   **数据源**: `GET /api/v1/strategy/strategies` (需包含今日盈亏字段)
*   **展示内容**:
    *   今日运行中策略列表。
    *   关键列: 策略名, 今日盈亏 (P&L), 持仓市值, 运行状态 (Running/Paused)。
*   **视觉**: 盈利策略行背景微红 (A股涨)，亏损行背景微绿。

## 3. 交互设计
*   **自动刷新**:
    *   行情数据: WebSocket 实时推送 (或 3s 轮询)。
    *   预警数据: WebSocket 实时推送。
*   **快捷操作**:
    *   每个卡片右上角有 "More >" 链接指向对应详情页。

## 4. API 映射汇总
| 组件区域 | 核心 API 端点 | 请求方式 | 备注 |
| :--- | :--- | :--- | :--- |
| 核心指数 | `/api/v1/market/quotes` | GET/WS | 需指定指数代码 |
| 资金流向 | `/api/v1/market/fund-flow` | GET | 包含北向和行业资金 |
| 实时预警 | `/api/v1/monitoring/alerts` | GET | 默认显示未处理(active) |
| 策略概览 | `/api/v1/strategy/strategies` | GET | 需聚合今日表现数据 |
