## 🚨 应急响应

### 一级故障（系统不可用）
1. 立即启动 **[应急响应预案.md](./应急响应预案.md)**
2. 30分钟内响应，2小时内恢复
3. 故障恢复后24小时内提交故障报告

### 二级故障（功能异常）
1. 参考 **[运维常见问题手册.md](./运维常见问题手册.md)** 处理
2. 2小时内响应，8小时内修复
3. 修复后更新问题手册

### 三级故障（性能下降）
1. 使用 **[系统性能优化指南.md](./系统性能优化指南.md)** 优化
2. 4小时内响应，24小时内优化
3. 更新 **[运维效果分析报告.md](./运维效果分析报告.md)**
