## 1. 资金流向趋势分析

### 功能概述
查看行业资金流向数据，点击行业名称可查看该行业的历史趋势图表。

### 使用步骤

1. **访问页面**：导航至 `市场数据 > 资金流向`

2. **浏览表格**：
   - 表格显示各行业的资金流向数据
   - 表头固定：滚动表格时，表头保持可见
   - 斑马纹样式：便于阅读

3. **调整分页**：
   - 页面右上角：选择每页显示条数（10/20/50/100）
   - 页面底部：翻页控件
   - 设置会自动保存到浏览器

4. **查看趋势图**：
   - 点击任意行业名称（蓝色可点击链接）
   - 表格下方显示该行业的资金流向趋势图
   - 图表显示：净流入、主力流入、散户流入三条曲线
   - 再次点击其他行业可切换图表

### 性能指标
- 图表更新时间：< 500ms
- 点击响应即时

## 2. 大数据量表格浏览

### 功能概述
ETF 行情和龙虎榜页面增强了表格浏览体验，支持固定表头和灵活分页。

### 适用页面
- `市场数据 > ETF行情`
- `市场数据 > 龙虎榜`

### 使用方法

1. **固定表头滚动**：
   - 向下滚动表格时，表头自动固定在顶部
   - 无需返回顶部即可查看列名

2. **分页控制**：
   - 右上角：选择每页显示数量（10/20/50/100）
   - 底部：分页导航
   - 当数据少于一页时，分页控件自动隐藏

3. **数据排序**：
   - 点击列头可按该列排序
   - 支持升序/降序切换

## 3. 自选股组织管理

### 功能概述
使用选项卡方式组织自选股，支持 4 种分组：用户自选、系统自选、策略自选、监控列表。

### 使用步骤

1. **访问页面**：导航至 `自选股`（原"股票管理"）

2. **切换分组**：
   - 点击顶部选项卡：用户自选 / 系统自选 / 策略自选 / 监控列表
   - 每个选项卡显示该分组的股票列表
   - 选项卡带有图标，便于识别

3. **查看股票**：
   - 表格显示股票代码、名称、分组、最新价、涨跌幅等信息
   - 分组标签：不同颜色区分不同子分组
   - 固定表头：滚动时表头保持可见

4. **状态持久化**：
   - 刷新页面：自动恢复上次查看的选项卡
   - 浏览器后退/前进：选项卡状态同步 URL 参数

5. **股票操作**：
   - 查看：点击"查看"按钮查看股票详情
   - 移除：从当前分组中移除股票

### URL 参数
可直接访问特定分组：`/watchlist?tab=strategy`（访问策略自选）

## 4. 全局字体大小调整

### 功能概述
支持 5 级字体大小调整，所有页面统一应用，设置立即生效。

### 使用步骤

1. **访问设置**：导航至 `系统设置 > 显示设置`

2. **选择字体大小**：
   - 特小 (12px)
   - 小 (14px)
   - 中 (16px) - 默认
   - 大 (18px)
   - 特大 (20px)

3. **预览效果**：
   - 设置卡片下方显示实时预览
   - 展示不同层级文字效果（辅助文字、正文、副标题、标题、大标题）

4. **应用设置**：
   - 选择后立即生效，无需刷新页面
   - 全局应用到所有页面、所有组件
   - 设置自动保存到浏览器

### 字体层级
系统使用动态字体层级，基于选择的基础字号自动计算：
- 辅助文字 = 基础字号 - 2px
- 正文 = 基础字号
- 副标题 = 基础字号 + 2px
- 标题 = 基础字号 + 4px
- 大标题 = 基础字号 + 6px

### 性能
- 字体切换时间：< 200ms
- 立即更新，无闪烁

## 5. 问财筛选快速查询

### 功能概述
提供 9 个预设查询，一键执行常用股票筛选条件。

### 使用步骤

1. **访问页面**：导航至 `市场数据 > 问财筛选`

2. **选择预设查询**：
   页面顶部显示 9 个查询卡片（3 行 x 3 列）：
   - **qs_1**: 高市值蓝筹股
   - **qs_2**: 连续上涨股
   - **qs_3**: 低估值股票
   - **qs_4**: 科技成长股
   - **qs_5**: 高股息率股票
   - **qs_6**: 突破新高股
   - **qs_7**: 医药健康股
   - **qs_8**: 超跌反弹股
   - **qs_9**: 高ROE优质股

3. **执行查询**：
   - 点击任意查询卡片
   - 卡片显示加载状态
   - 下方表格自动显示查询结果

4. **查看结果**：
   - 表格显示：序号、股票代码、名称、最新价、涨跌幅等
   - 涨跌幅：红色（上涨）/ 绿色（下跌）
   - 支持列排序
   - 支持分页浏览

5. **切换查询**：
   - 点击其他查询卡片
   - 之前的结果自动清空
   - 显示新的查询结果

### 性能
- 查询响应时间：< 1秒
- 结果立即展示

### 自定义查询
除预设查询外，还可使用顶部输入框进行自定义文本查询。

## 💡 使用技巧

### 表格浏览
- 使用固定表头功能，无需频繁滚动回顶部
- 调整分页大小，根据屏幕尺寸优化显示
- 浏览器会记住你的分页偏好

### 字体设置
- 推荐字体大小：
  - 桌面显示器：16px (中) 或 18px (大)
  - 笔记本：14px (小) 或 16px (中)
  - 高分辨率屏幕：18px (大) 或 20px (特大)

### 自选股管理
- 使用浏览器书签保存常用分组页面（如 `/watchlist?tab=strategy`）
- 利用分组颜色标签快速识别不同股票组

### 问财筛选
- 先浏览预设查询了解系统筛选能力
- 参考预设查询编写自定义查询语句
- 查询结果可导出 CSV 格式
