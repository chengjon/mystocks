# MyStocks 量化交易平台 - Web页面结构详细描述

## 概述

MyStocks是一个现代化的量化交易平台，采用Vue 3 + TypeScript + Element Plus + ArtDeco设计系统技术栈，结合FastAPI后端，提供完整的量化交易、策略回测、实时监控等功能。整体设计风格为专业金融风，深色主题优先，数据可视化为核心。

系统采用模块化架构，包含多个功能域：
- **仪表盘域**: 市场概览、投资组合监控
- **市场域**: 实时行情、技术分析、资金流向
- **交易域**: 订单管理、持仓监控、交易历史
- **策略域**: 策略创建、回测分析、信号生成
- **风险域**: 风险监控、公告监控、告警管理
- **监控域**: 系统监控、数据质量、性能分析
- **设置域**: 系统配置、主题设置、安全管理

---

## 核心设计规范

### ArtDeco设计系统
- **色彩系统**: 金色主色调(#D4AF37)，红色上涨(#FF5252)，绿色下跌(#00E676)
- **组件库**: 52个专用ArtDeco组件，包含按钮、卡片、图表、表单等
- **视觉风格**: 几何装饰边框、渐变背景、分层阴影效果
- **响应式**: 支持桌面端专业显示，移动端适配优化

### 布局架构
- **多层级路由**: 主布局 → 功能域布局 → 页面组件
- **嵌套导航**: 面包屑导航 + 标签页切换 + 侧边栏菜单
- **状态管理**: Pinia状态管理，组合式API架构
- **组件复用**: 通用组件库 + 业务组件分层

### 交互模式
- **实时更新**: SSE/WebSocket双重推送，自动刷新机制
- **智能加载**: 骨架屏加载，错误重试，缓存策略
- **操作反馈**: 悬停效果、加载状态、成功/失败提示
- **数据导出**: CSV/Excel格式导出，批量操作支持
