## 整体架构

### 技术栈
- **前端框架**: Vue 3.5.22 (Composition API)
- **UI 组件库**: Element Plus 2.11.4
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **图表库**: ECharts 5
- **构建工具**: Vite 5.4.20
- **Node 版本**: 20.19.5 LTS

### 项目结构
```
src/
├── api/              # API 接口封装
├── assets/           # 静态资源
├── components/       # 公共组件 (目前为空)
├── layout/           # 主布局组件
├── router/           # 路由配置
├── stores/           # Pinia 状态管理
├── views/            # 页面组件
└── App.vue          # 根组件
```

### 路由结构
- `/login` - 登录页面
- `/` - 主布局 (需要登录)
  - `/dashboard` - 仪表盘 (默认首页)
  - `/market` - 市场行情
  - `/stocks` - 股票管理
  - `/analysis` - 数据分析
  - `/settings` - 系统设置

## 附录

### 后端 API 列表
当前可用的 API 接口:

1. **认证相关**:
   - `POST /api/auth/login` - 用户登录
   - `POST /api/auth/logout` - 用户登出
   - `GET /api/auth/me` - 获取当前用户
   - `POST /api/auth/refresh` - 刷新令牌

2. **数据相关**:
   - `GET /api/data/stocks/basic` - 获取股票基本信息
   - `GET /api/data/stocks/daily` - 获取日线数据
   - `GET /api/data/markets/overview` - 获取市场概览
   - `GET /api/data/stocks/search` - 搜索股票
