A股市场颜色 (A-Share Market Colors)

中国大陆 A 股市场使用红涨绿跌：

↑ 涨 (UP) - 红色 #FF5252
↓ 跌 (DOWN) - 绿色 #00E676
→ 平 (FLAT) - 灰色 #B0B3B8

使用说明 (Usage Instructions)

在 Vue 组件中使用这些 CSS 变量：

<style scoped>
.stock-price {
  color: var(--color-up);  /* 红色 - 涨 */
}

.stock-price.down {
  color: var(--color-down);  /* 绿色 - 跌 */
}

.card {
  background-color: var(--bg-card);
  color: var(--text-primary);
}
</style>
